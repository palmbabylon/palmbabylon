# نخلة بابل - Palm of Babylon

تم إعداد هذا الموقع ليعمل مباشرة على GitHub Pages.

## الملفات الموجودة
- `index.html` (الصفحة الرئيسية)
- `.nojekyll` (لتجاوز فلتر Jekyll على GitHub Pages)
- `README.md` (هذا الملف)

## تغيير شعار الموقع
1. ضع الصورة التي تريد استخدامها كشعار داخل مجلد `public` في المشروع أو في نفس المجلد الجذر.
2. تأكد من اسم الملف: `palm-of-babylon-logo.png`.
3. إذا وضعت الصورة في `public`، تأكد من أنها تُعرض عبر هذه المسار:
   - `src="/palm-of-babylon-logo.png"`

## خطوات النشر على GitHub Pages
1. افتح مستودع جديد على GitHub.
2. ادفع (`git push`) محتويات المجلد إلى الفرع `main` أو `master`.
3. في إعدادات المستودع (Settings) > Pages:
   - اختَر الفرع `main` (أو `master`) والمجلد `/` (root).
   - احفظ.
4. بعد بعض الثواني، انتقل إلى الرابط المعروض (https://<username>.github.io/<repo-name>).

## إصلاحات تم إنجازها
- تم تحديث جميع مراجع الشعار (`header`, `hero`, `footer`) إلى `palm-of-babylon-logo.png`.
- تم ضمان أن روابط التنقل والانتقال السلس تعمل.
- تم إضافة `.nojekyll` لدعم GitHub Pages.
- تم فحص التسلسل الهرمي لعلامة HTML وتحقق من سلامة البناء.
